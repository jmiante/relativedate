Instalação

Para Instalar a biblioteca RelativeDate, basta abrir o terminal e instalar via pip, digitando o seguinte comando:

pip install relativedate
Criando um objeto RelativeDate

Um objeto RelativeDate pode ser instanciado de duas formas, sendo:

1. Instanciando sem argumento, isso irá gerar um objeto RelativeDate com o atributo datetime com a data e hora do momento que instanciar o Objeto para isso, basta digitar o código abaixo:

from relativedate import RelativeDate

rd = RelativeDate()
print(rd.datetime)

RETORNO:
2023-05-07 21:00:00

*** Supondo que foi o objeto foi instanciado no dia 23/05/2023 às 21:00hs

2. Instanciando a partir de um datetime, o objeto RelativeDate pode receber como argumento um datetime, com isso será gerado o objeto RelativeDate com o atributo datetime com os dados do objeto datetime passado no argumento, vamos para o exemplo abaixo:

from datetime import datetime
from relativedate import RelativeDate

dt = datetime(year=2023, month=5, day=5)
rd = RelativeDate(dt)
print(rd.datetime)

RETORNO:
2023-05-05 00:00:00
Função addYear(relative_year)

A função addYear irá alterar o ano do atribuito datetime do objeto RelativeDate de acordo com o parametro relative_year, segue abaixo um exemplo de uso:

from datetime import datetime
from relativedate import RelativeDate

dt = datetime(year=2023, month=5, day=5)
rd = RelativeDate(dt)
print(rd.datetime)

rd.addYear(-1)
print(rd.datetime)

rd.addYear(3)
print(rd.datetime)

RETORNO:
2023-05-05 00:00:00
2022-05-05 00:00:00
2025-05-05 00:00:00

Função add(year, month, day, hour, minute, second)

A função add irá alterar os atributos year, month, day, hour, minute e second do datetime do objeto RelativeDate de acordo com o parametros acima, nenhum parâmetro é obrigatório, em caso de não explicitar o mesmo será considerado com o valor de 0, segue abaixo um exemplo de uso:

from datetime import datetime
from relativedate import RelativeDate

dt = datetime(year=2022, month=9, day=2)
rd = RelativeDate(dt)
print(rd.datetime)

rd.add(year=10, month=1, day=5)
print(rd.datetime)

RETORNO:
2022-09-05 00:00:00
2023-10-07 00:00:00
Função dateDiff(datetime)

A função dateDiff irá retornar um objeto timedelta com a distância do atributo datetime do objeto RelativeDate com o do parâmetrodatetime passado na função, segue abaixo um exemplo de uso:

from datetime import datetime
from relativedate import RelativeDate

dt = datetime(year=2022, month=9, day=2)
dt2 = datetime(year=2023, month=5, day=20)

rd = RelativeDate(dt)
print(rd.datetime)
print(dt2)

date_diff = rd.dateDiff(dt2)
print(date_diff)
RETORNO:
2022-09-02 00:00:00
2023-05-20 00:00:00
260 days, 0:00:00
